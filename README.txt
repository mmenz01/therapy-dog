therapyDog Google Chrome Extension

A virtual therapy dog for a moment of instant joy and calm while working online!

To install therapyDog to Google Chrome (it has not yet been published to the Chrome Web Store):
1. Download therapyDog.zip and unzip it.
2. Within your Chrome browswer, go to chrome://extensions/ and enable "Developer mode" (a button in the top right corner).
3. Click "Load unpacked" and select the therapyDog folder.
4. The extension is now installed! Pin it to your Chrome toolbar for easy access!
